> A clone of Bomberman made in C++, using Raylib, Boost, featuring custom Raylib encapsulation and an Entity-Component System

## Install

```sh
mkdir build && cd build && cmake --build . && cd ..
```

## Usage

```sh
./bomberman [-f : start fullscreen (default : windowed)]
```
